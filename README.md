# Blue vs Red: Notebook War 3D

## README

### Game Overview

**Blue vs Red: Notebook War 3D** is a stylized, interactive 3D first-person shooter where you control a Blue Stickman fighting Red Stickmen in immersive notebook-inspired environments. The game features fast-paced combat, multiple weapons, AI-driven enemies, and dynamic levels designed to challenge your tactical skills.

### Key Features

* **Unique Visual Style:** Black-and-white hand-drawn notebook aesthetic with low-poly characters and sketch-style VFX.
* **Interactive 3D FPS Gameplay:** Move, jump, sprint, crouch, slide, climb, and interact with objects.
* **Multiple Levels:** Apartment complex, hotel siege, and jungle sketchfield, each with unique objectives and bosses.
* **Diverse Weapons:** Pistols, SMGs, assault rifles, shotguns, sniper rifles, and grenades with realistic mechanics.
* **Enemy AI:** Red Stickmen with patrol, chase, flank, and retreat behaviors, including elite units and bosses.
* **Pickups & Boosts:** Ammo, medkits, armor, speed, and damage boosts.
* **Cross-Platform Support:** PC (keyboard + mouse) and mobile (touch controls).
* **Supabase Backend:** Cloud save, authentication, leaderboard, inventory management, and player profiles.

### Installation & Setup

1. Clone or download the Unity project.
2. Open in Unity 2022 LTS or newer.
3. Configure Supabase backend with your project keys.
4. Build and run for your target platform (PC, Android, or iOS).
